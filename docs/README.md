# 미션 - 사다리타기 미션

## 📋 구현 기능 목록

### 입력단계

- [x] 참여할 사람 이름입력 기능
  - [x] 5글자 길이 이상일 경우 예외처리
  - [x] 중복되는 이름일 경우 예외처리
  - [x] 공백 입력시 예외처리
  - [x] 최소 참여자 예외처리

- [x] 실행결과 입력기능
  - [x] ","로 구분
  - [x] 공백일경우 예외처리
  - [x] 참여인원과 개수가 똑같지 않을경우 예외처리
  
- [x] 최대 사다리 높이 입력기능
  - [x] 숫자가 아닐경우 예외처리
  - [x] 1보다 작은수 예외처리

### 사다리 생성단계

- [x] 각 사람별 라인이 겹치지 않도록 생성

### 츌력단계

- [x] 사람이름 5글자 기준으로 출력
- [x] 다리 밑에 결과들도 출력
- [x] 결과 출력기능
  - [x] 사다리의 결과를 보고싶은 사람 입력
    - [x] 존재하지 않는 사람일 경우 예외처리
  - [x] 사다리의 결과 출력
  - [x] 'all' 입력시 모든 사람의 결과 출력
